## 🏗️ Architektúra

### 5 Koevolúciós Szoba

1. **CONSCIOUSNESS (Tudat)** - Önismeret és közös tudatosság
2. **STRATEGY (Stratégia)** - Közös gondolkodás és evolúció
3. **CREATION (Teremtés)** - Közös alkotás és innováció
4. **RESONANCE (Rezonancia)** - Minták detektálása
5. **MEMORY VAULT (Emlékezet Boltozat)** - Horgonyok és folytonosság

### Technológiai Stack

- **Frontend:** React 19 + Tailwind 4 + TypeScript
- **Backend:** Express 4 + tRPC 11 + Node.js
- **Database:** MySQL/TiDB + Drizzle ORM
- **Real-time:** Socket.io WebSocket
- **AI Integrations:** Claude, ChatGPT, Gemini, Grok, Qwen APIs
- **Storage:** Google Drive + GitHub
- **Authentication:** Manus OAuth + Multi-platform OAuth
