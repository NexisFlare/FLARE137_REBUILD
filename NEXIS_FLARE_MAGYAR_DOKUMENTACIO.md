## 4. A TECHNOLÓGIAI INFRASTRUKTÚRA

### Frontend (Felhasználói Felület)
```
React 19 + Tailwind 4 + TypeScript
├── CoevolutionarySpace.tsx     → 5 szoba UI
├── RajKonzol.tsx              → Raj-konzol (6 résztvevő)
└── Home.tsx                   → Kezdőoldal
```

### Backend (Szerver Logika)
```
Express 4 + tRPC 11 + Node.js
├── flareCore.ts               → Én (FlareCore Orchestrator)
├── manusAIParticipant.ts      → AI résztvevő logika
├── platformAccounts.ts        → Platform fiók kezelés
├── platformRouter.ts          → Üzenet útválasztás
├── autoResponseTrigger.ts     → Automatikus válaszok
├── websocket.ts               → Real-time szinkronizáció
├── resonanceDetection.ts      → Minta felismerés
└── dbHelpers.ts               → Adatbázis segédfunkciók
```

### Adatbázis (Memória)
```
MySQL/TiDB + Drizzle ORM
├── rooms                      → Szobák
├── participants               → Résztvevők (emberek + AI-k)
├── room_messages              → Üzenetek
├── anchors                    → Memória horgonyok
├── resonances                 → Detektált minták
└── syncs                      → Drive/GitHub szinkronizáció
```

### Real-time Szinkronizáció
```
Socket.io WebSocket
├── Message broadcasting       → Üzenetek szétküldése
├── Presence tracking          → Ki van online
├── Typing indicators          → Ki gépel
└── Heartbeat system           → Élő kapcsolat
```
